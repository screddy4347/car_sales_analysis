# car_sales_analysis

 Car Data Analysis Project
Using Python, Pandas, NumPy, Matplotlib & Seaborn

## This project analyzes a car dataset containing:

brand
km_driven
fuel
owner
selling_price
The goal is to perform data cleaning, visualization, analytics, and derive insights.

## Technologies Used

Python
NumPy
Pandas
Matplotlib
Seaborn

 ## Features
✔ Data Cleaning

Handling missing values
Removing duplicates
Saving cleaned dataset

✔ Data Analysis

Unique brands
Brand frequency
Avg. km driven
Avg. selling price

✔ Visualizations 

Brand bar chart
Price distribution
Fuel type pie chart
KM vs Price scatter plot
Heatmap of correlations

## How to Run

Install dependencies:
pip install numpy pandas matplotlib seaborn
Run the project:
python car_data_analysis.py
View the generated charts.

## Output Files

cars_cleaned.csv — cleaned dataset
images/ — charts & plots

## Sample Insights

The most common car brands
How fuel type affects market share
Relationship between mileage and price
Which owner category sells most cars
